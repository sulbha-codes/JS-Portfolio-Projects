# 💼 LoanLogic — Loan Eligibility Checker  


**LoanLogic** is a mini web project built with **HTML, CSS, and JavaScript** that determines a user’s loan eligibility based on key financial inputs — **age, income, credit score, and loan amount**.  

It showcases practical use of **JavaScript conditionals**, **form validation**, and **responsive UI design**, reflecting strong frontend logic and problem-solving skills.

## 🧠 Project Overview  

LoanLogic helps users check their **loan eligibility** in seconds by analyzing financial details through predefined logical conditions — similar to real loan approval systems.  
It provides instant feedback with clear, color-coded results and a minimal, professional interface.  

### 💡 How It Works  
1. User enters: **Age**, **Monthly Income (₹)**, **Credit Score**, **Desired Loan Amount (₹)**  
2. Clicks **“Check Eligibility”**  
3. The app evaluates inputs and displays results:  
   - ✅ **Approved** — All criteria met  
   - ⚠️ **Conditional** — Needs co-signer or lower loan amount  
   - ❌ **Denied** — Age or credit score not eligible

## 🧮 Eligibility Rules  

| Criteria | Condition | Result |
|-----------|------------|--------|
| Age | 21–60 years | ❌ Denied if outside range |
| Credit Score | ≥ 600 | ❌ Denied if lower |
| Monthly Income | ≥ ₹30,000 | ⚠️ Conditional if lower |
| Loan Amount | ≤ 10× income | ⚠️ Conditional if higher |
| All valid | Meets all criteria | ✅ Approved |
